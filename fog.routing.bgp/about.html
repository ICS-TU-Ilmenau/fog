Basis/Dependencies

This software uses the BGP implementation of the Scalable Simulation Framework (SSF) 2.0 (from 2004) as a basis. The implementation was adapted for FoG and, thus, uses different interfaces (e.g. for setting up connections). However, the internal BGP logic was not modified. The dependencies to software from Raceway have been removed.

In addition, the BGP code depends on the Regexp project, which is included as JAR.
